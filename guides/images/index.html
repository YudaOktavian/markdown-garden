Images

Images are pictures that are intended to be embedded in the current document, from either the same server or from an external image provider.

In Markdown, images can be defined using similar syntax to links but by preceding the markup with an exclamation mark.

Images are defined in two ways: inline images and reference images.

  • Inline Images
  • Reference Images

Both inline links and reference links allow the writer to define alternative text to be rendered on the page if the image is unavailable or to web browsers and screen readers which cannot display images.
